**APP DESCRIPTION**
Pixelion is an advanced, feature-rich application tailored for image conversion and resizing needs. It caters to a wide range of users, from hobbyists to industry professionals, by providing seamless support for diverse file formats and cutting-edge functionalities.

**Key Features:**

- **Comprehensive Format Support:** Convert and resize popular formats like JPEG, PNG, WEBP, AVIF, TIFF, HEIC, NEF, PDF, PSD, AI, EPS, SVG, and more.
- **Advanced Animation Handling:** Extract and process individual frames from animated WEBP and GIF files.
- **Folder & Subfolder Processing:** Maintain original folder structures while processing files in batch mode.
- **Live Progress Updates:** Monitor real-time conversion progress with live counters displaying processed and total files.
- **Custom Output Options:** Choose custom dimensions, maintain aspect ratios, and set quality levels for precise results.
- **Dynamic Notifications:** Receive sound alerts and pop-up notifications for successful operations, warnings, and errors.
- **Optimized Error Handling:** Clear guidance on unsupported formats or corrupted files, ensuring minimal disruptions.
